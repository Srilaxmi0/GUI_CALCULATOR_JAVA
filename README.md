# Java AWT Calculator

This is a simple calculator application built using Java AWT (Abstract Window Toolkit). The calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division. It includes a clear button to reset the input field.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button to reset the input field.
- Color-coded user interface for better usability.

## User Interface

- **TextField**: Displays the input and result.
- **Buttons**: For digits (0-9), operators (+, -, *, /), equals (=), and cancel.
- **Panel**: Organizes buttons in a 4x4 grid.
- **Frame**: Main window for the application.

### Button Colors

- Number buttons (0-9): Light gray
- Operator buttons (+, -, *, /): Orange
- Equals button (=): Green
- Cancel button: Red
